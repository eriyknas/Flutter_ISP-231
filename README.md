# Лабораторная работа №1. Быстрое погружение в язык Dart
Консольное приложение для управления задачами, созданное в рамках изучения основ языка Dart.
Проект представляет собой менеджер задач с поддержкой добавления, отметки выполнения, удаления и просмотра списка. Данные хранятся в памяти в течение сессии.

## Ответы на вопросы
1. final и const в Dart оба запрещают переназначать переменную, но отличаются временем инициализации и областью применения.
2. Переменная может хранить либо строку (String), либо значение null.
3. · Обычное значение – доступно синхронно.
   · Future<T> – представляет результат операции, которая завершится позже.
   · await приостанавливает выполнение текущей async-функции, не блокируя основной поток.
4. Перегрузка конструкторов в C# – разные наборы параметров, но одно и то же имя класса.
В Dart перегрузка конструкторов невозможна.
